# מוטיבי מותג למצגת

נכסים שנגזרו מתוך המיתוג שסופק וישמשו את המצגת האינטראקטיבית והחומרים העתידיים.

| מוטיב | קובץ | שימוש מומלץ |
|---|---|---|
| קו מסילה עם נקודות | נבנה ב-CSS/SVG בתוך `index.html` | שקפי פתיחה, מעבר וסיום |
| גריד נקודות חול | נבנה ב-CSS כ-`.dot-grid-motif` | שכבת רקע עדינה בשקפי תוכן |
| פס מותג תחתון | `gaolim-brand-footer-strip.png` | שקפי תוכן / חלוקה לפרקים / עמודים שבהם צריך חתימת מותג חזקה |
| איור חוץ / קו שכונה | `gaolim-skyline-official-transparent.png` | תחתית שקפים נקייה, בלי הפס הכחול והטקסטים |

הנחיית שימוש: לשלב את פס המותג התחתון לסירוגין, לא בכל שקף, כדי שלא יעמיס. הוא מתאים במיוחד לשקפים שמסכמים עוגנים, מציגים פרקים, או סוגרים חלק במצגת.
