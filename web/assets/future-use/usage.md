# נכסים לשימוש עתידי במצגת

## school-entrance-signage-mockup.png

הדמיית שילוט כניסה לבית הספר.

שימוש מומלץ:
- לקראת סוף המצגת, בפרק "איך זה ייראה בבית הספר?"
- להמחשת המעבר מרעיון חינוכי לשפה פיזית במרחב.
- מתאים במיוחד אחרי שקפי השפה העיצובית ולפני הפיילוט השנתי.

מסר לשקף:
"כך זה יכול להיראות בשער בית הספר: השפה יוצאת מהמצגת אל המרחב שבו הילדים, ההורים והקהילה פוגשים את גאולים בכל בוקר."
