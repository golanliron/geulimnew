# אייקונים למצגת גאולים המסילה

האייקונים נשמרו כקבצים נפרדים כדי שאפשר יהיה להשתמש בהם בשקפים, באתר ובחומרי המשך בלי לחתוך בכל פעם מחדש.

| אייקון | קובץ | שימוש מומלץ |
|---|---|---|
| למידה מבוססת מקום | `place-based-learning.png` | חזון, PBE, השכונה ככיתה |
| חקר וגילוי | `research-discovery.png` | מחקר, מדעים, חקר שטח |
| קהילה | `community.png` | קהילה כשותפה, הורים ושכונה |
| יוזמות | `initiative.png` | פיילוט, רעיונות, יום מסילה |
| מנהיגות | `leadership.png` | מסוגלות, אחריות והשפעה |
| קיימות | `sustainability.png` | טבע עירוני, סביבה ופארק המסילה |
| שכונה | `neighborhood.png` | בקעה, מקום וחיבור למרחב |
| צמיחה אישית | `personal-growth.png` | רווח לילדים, מסע אישי והתפתחות |

הנחיית שימוש: לשמור על האייקון בתוך עיגול השמנת, עם כותרת קצרה מתחתיו או לצידו. לא למתוח את האייקונים, ולא לערבב צבעים מחוץ לפלטת המותג.
