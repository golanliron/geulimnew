# קמפוס גאולים המסילה

מצגת אינטראקטיבית וספר מותג חי עבור קמפוס גאולים המסילה.

## פתיחה מקומית

```powershell
cd web
python -m http.server 4173 --bind 127.0.0.1
```

ואז לפתוח:

- מצגת דוגמה: `http://127.0.0.1:4173/`
- ספר מותג: `http://127.0.0.1:4173/brand.html`

## מבנה

- `web/index.html` - דוגמת מצגת אינטראקטיבית.
- `web/brand.html` - ספר מותג HTML.
- `web/assets/` - לוגואים, לוחות מקור ומוטיבים.
- `web/assets/brand-elements/` - אייקונים ומוטיבים כקבצים נפרדים על רקע שקוף.
